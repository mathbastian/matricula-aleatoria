# Análise dos Resultados
 
À medida que o número máximo de cada cenário aumenta, mais o sistema erra pois existem mais possibilidades.
É possível perceber que cenário 5 possui o maior número de números rejeitados disparado.
A recomendação é que se limite o número máximo para 10, como no cenário 10, pois como a comparação com o número randômico é feita número a número da matrícula, 9 é o maior número possível a ser gerado. Como consequência, menos números serão rejeitados.
